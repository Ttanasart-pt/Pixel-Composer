Surface replace node search for subsurfaces in a surface and replace with a new one.

<img surface_replace/>

## Properties

### <junc array mode/>

When the <junc target image/> and the <junc replacement image/> is an array. You can set whether to match 
each target to a replacement, or to randomize the replacement image index.

### <junc fast mode/>

Fast mode match only the origin point of the image instead of the entire image. This results in faster 
processing but may not be as accurate.

### <junc color threshold/>

The threshold used to differentiate two color.

### <junc pixel threshold/>

The amount of pixels need to be match to consider a subsurface.

### <junc draw base image/>

Whether to draw original image behind the replacement.
