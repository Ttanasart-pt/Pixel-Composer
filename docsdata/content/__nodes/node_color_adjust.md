Color adjust node is use for color grading.



## Properties


### <junc brightness/>


Adjust the brightness of the image by adding brightness value uniformly across the surface.


### <junc exposure/>


Adjust the exposure of the image by multiplying the exposure value to the pixel. This make the dark pixel 
stays dark and the bright pixel becomes brighter.


### <junc contrast/>


Adjust the contrast of the image, the different between darker and brighter colors.


### <junc hue/>


Change the hue of the image.


### <junc saturation/>


Adjust the saturation of the image.


### <junc value/>


Adjust the value of the image.



## Blend


Blending properties apply <junc blend/> color to a pixel based on the <junc blend mode/> and 
<junc blend amount/>.


The detail on blend mode can see in <node blend/>.
