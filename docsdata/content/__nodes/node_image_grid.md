Image grid is a grid version of <node stack/>. It allows a composition of multiple images in a grid form.



## Grid Properties


### <junc Main Axis/>


<img grid_main_axis>


The main axis where the next surface will be placed.


### <junc Column/>


<img grid_column>


The number of surfaces in the main axis before the surface moved to the next line.


### <junc Spacing/>


<img grid_spacing>


The spacing between each surface.
