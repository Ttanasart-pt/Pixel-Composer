Non-uniform Blur (formerly simple blur) is a blur node that use simpler algorithm than the <node blur/>, which 
allows you to control the blur size per pixel.

<img-deco blur_simple caption="Credit: Synth Cities Environment by ansimuz">

## Properties

### <junc size/>

Maximum blur size.

### <junc blur mask/>

Extra surface use for scaling blur effect.

### <junc override color/>

Replace the resulting color with a uniform color, use when you only want to blur the alpha channel.

### <junc gradient/>

Apply color blending per sampling.
