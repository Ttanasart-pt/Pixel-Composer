Point light is a type of light that illuminate surrounding area in a spherical shape. Unlike directional light 
point light get dimmer as it goes further from the source.

## Shadow


Selecting "Cast Shadow" will make the light cast shadow with solid objects. Note that shadow does not works with 
transparent and translucent objects.


Shadow Map Size is used to control shadow quality. Shadow Bias can be used to fix glitch with light
bleed around the shadow edges.
