Trail node draw line to connect pixel between each frames.



## Properties


### <junc max life/>


The amount of frame to draw trail.


### <junc loop/>


Whether to loop back to the last frame in the beginning.


### <junc max distance/>


Maximum object movement speed. Adjust this value to decrease processing time and artifact.


### <junc match color/>


Whether to only match pixel with the same color.


### <junc alpha over life/>


Modulate alpha over time.
